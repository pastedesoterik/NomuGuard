진짜 말 그대로 64비트 프로그램의 싱글 스레드용 메모리 보호 도구

부호있는 int, long long, float만 지원

자세한 사용법은 NomuGuard.h에 명시되어있음

의도된 사용은 lib로 컴파일하고 프로젝트에 lib와 헤더를 추가해서
코드에 녹여 사용하는것이지만 어떻게 쓰든 사실 큰 상관은 없음

요약: 메모리를 xor 암호화 후 PAGE_NOACCESS로 권한을 변경해서 보호
상호작용 할때마다 커널을 왔다갔다 하기때문에 보안을 약간 포기해서 성능을 극대화 시키려면
Protect함수에 level 매개변수에 1을(사실 2만 아니면되지만 1로 하는게 미래의 당신에게 이로울 예정) 넣으면
xor 암호화만 진행해서 성능을 챙길수있음

커널 드라이버 가져다 깽판치는건 못막더라도
치트엔진 깔짝이는것들은 다 쳐냄

C#에서도 사용가능
dll로 컴파일하거나 dll을 받아서
C# 프로젝트에 넣고 그 dll을 참조에 추가
using NomuGuard_CS_library;를 사용하면 사용 가능
